# AN SDK Examples

Example projects for the [AN SDK](https://an.dev) — deploy AI coding agents and connect them to your app.

## Examples

| Example | Description |
|---------|-------------|
| [`nextjs`](./nextjs) | Next.js app — agent definition + token handler + chat UI |

## Quick Start

```bash
cd nextjs
npm install
npx an login
npx an deploy
cp .env.example .env.local
npm run dev
```
